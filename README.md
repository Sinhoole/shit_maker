## 📖 核心理念 (Manifesto)

本项目旨在赋予每个人**生产“大份”**的能力。

在这个信息爆炸的时代，我们需要更多、更大、更纯粹的内容。无论它是文学巨著，还是互联网排泄物，这里都是它的生产车间。

在这里，逻辑的枷锁被打破，伦理的边界由你定义：

*   想要看 **安倍晋三** 与 **山上彻也** 在异世界重逢并组建乐队？**可以。**
*   想要看 **爱因斯坦** 与 **霍金** 在秋名山进行轮椅漂移对决？**没问题。**
*   想要模仿 **鲁迅** 的笔触描写 **赛博朋克 2077**？**安排。**

一切皆由你创造。启动机器，开始制造你的那一份“大份”吧。

## ✨ 功能特性 (Features)

*   **🧠 多核驱动 (Multi-Model Support)**: 
    *   支持 **Google Gemini** (原生集成), **OpenAI**, **DeepSeek**, **Aliyun Qwen**, **SiliconFlow** 等多种大模型。
    *   支持自定义 Base URL，兼容所有 OpenAI 格式接口。
*   **🎛️ 精密控制 (Precision Control)**:
    *   自定义 **题材、篇幅、视角**。
    *   设定 **核心冲突、高潮情节、特定关键词**。
    *   **文风模仿**: 指定作家风格或提供示例文本进行模仿。
*   **⚡ 实时流式传输 (Real-time Streaming)**: 看着你的大作逐字逐句诞生，享受生成的快感。
*   **🎨 硬核复古 UI (Retro Industrial Design)**: 
    *   New Brutalism (新野兽派) + Retro Industrial 风格。
    *   无干扰的沉浸式写作体验。
*   **🌍 多语言支持**: 中、英、日、韩、俄五种语言界面与生成支持。
*   **💾 本地优先**: API Key 仅存储于浏览器 LocalStorage，安全无忧。

## 🚀 快速开始 (Quick Start)

### 本地运行

1.  **克隆仓库**
    ```bash
    git clone https://github.com/Sinhoole/story-forge-ai.git
    cd story-forge-ai
    ```

2.  **安装依赖**
    ```bash
    npm install
    ```

3.  **启动开发服务器**
    ```bash
    npm run dev
    ```

4.  **构建生产版本**
    ```bash
    npm run build
    ```

### 部署 (Vercel)

本项目完美支持 Vercel 一键部署：

1.  Fork 本仓库。
2.  在 Vercel 中导入项目。
3.  点击 **Deploy**。

## ⚙️ 配置指南

1.  点击右上角的 **系统设置 (SYSTEM CONFIG)** 按钮。
2.  选择你的 AI 供应商 (Provider)。
3.  输入对应的 API Key。
    *   *注意：如果你使用的是国内模型（如 DeepSeek、通义千问），建议选择对应的预设或使用 Custom 模式。*
4.  点击保存，开始制造。
