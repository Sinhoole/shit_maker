# 🏭 大份制造机 (Story Forge AI) - 操作手册

欢迎来到 **大份制造机**。本手册将指导你如何部署这台机器，并教你如何调整参数以生产出最纯粹的“大份”。

---

## 🚀 第一章：部署指南 (Deployment)

本项目是纯前端应用（Static Web App），部署极其简单。推荐使用 **Vercel** 进行托管，完全免费且速度极快。

### 1. 准备工作
*   拥有一个 [GitHub](https://github.com/) 账号。
*   拥有一个 [Vercel](https://vercel.com/) 账号（可直接用 GitHub 登录）。

### 2. 获取代码
如果你是开发者：
1.  Fork 本仓库到你的 GitHub 账户。

如果你是普通用户：
1.  下载本项目代码。
2.  在 GitHub 上新建一个仓库，将代码上传上去。

### 3. Vercel 一键部署 (推荐)
1.  登录 Vercel Dashboard。
2.  点击 **"Add New..."** -> **"Project"**。
3.  在 **Import Git Repository** 列表中，找到 `story-forge-ai` (或者你命名的仓库)，点击 **Import**。
4.  **配置构建 (Build Configuration)**:
    *   Vercel 会自动识别框架为 `Vite`。
    *   **无需修改任何设置**。
    *   Build Command: `npm run build`
    *   Output Directory: `dist`
5.  点击蓝色按钮 **Deploy**。
6.  等待约 1 分钟，屏幕上撒花庆祝时，点击预览图即可访问你的专属制造机。

## 🎮 第二章：使用指南 (User Manual)

### 1. 启动能源 (配置 API Key)

机器启动需要燃料。由于本项目保护隐私，所有 API Key **仅存储在你浏览器的 LocalStorage 中**，绝不上传云端。

1.  点击界面右上角的黄色按钮 **"系统设置 (SYSTEM CONFIG)"**。
2.  **选择供应商 (AI Provider)**:
    *   **Google Gemini**: 强烈推荐。速度快，免费额度大，生成长文能力强。
    *   **DeepSeek / SiliconFlow**: 国产之光，逻辑性强，适合写复杂的剧情。
    *   **OpenAI / Claude**: 传统强者。
    *   **Custom**: 如果你有自己的 API 代理地址，选这个。
3.  **填入密钥 (API Key)**:
    *   点击输入框上方的 "GET KEY ->" 链接可直达申请页面。
4.  **选择模型 (Model)**:
    *   手动输入或选择。推荐 `gemini-1.5-flash` (速度快) 或 `gemini-1.5-pro` (质量高)。
5.  点击 **"SAVE DATA"** 保存。

### 2. 设定参数 (Input Parameters)

左侧控制台是你的操作台，这里决定了“大份”的味道。

#### 基础参数 (Primary)
*   **题材 (Genre)**: 决定世界观。是赛博朋克、古代武侠，还是童话故事？
*   **长度 (Length)**: 短篇适合段子，长篇适合史诗。
*   **视角 (Viewpoint)**: 第一人称更有代入感，第三人称更客观。

#### 核心数据 (Core Data) - **灵魂所在**
*   **角色 (Characters)**: 
    *   输入你想让其登场的人。越离谱越好。
    *   *例*：“林黛玉（倒拔垂杨柳的健身教练），伏地魔（没鼻子的相声演员）”。
*   **冲突 (Conflict)**: 
    *   故事的矛盾中心。
    *   *例*：“伏地魔想教林黛玉说相声，但林黛玉只想举铁”。
*   **高潮事件 (Climax)**: 
    *   **必填，这是出大份的关键**。详细描述画面。
    *   *例*：“林黛玉用倒拔垂杨柳的神力把霍格沃茨连根拔起，用来当哑铃练二头肌”。
*   **关键词 (Keywords)**: 强制 AI 包含的词汇。

#### 风格调教 (Style)
*   **模拟作家 (Mimic Author)**: 
    *   让 AI 被魂穿。
    *   *例*：“鲁迅”（会用很多“大约、的确”），“古龙”（短句，装X），“郭敬明”（华丽辞藻）。
*   **风格样本 (Style Sample)**: 
    *   直接粘贴一段你喜欢的文字（比如《大悲咒》或者某篇营销号文章），AI 会模仿其句式写故事。

### 3. 生产与输出 (Output)
1.  点击下方巨大的 **"启动制造程序 (INITIATE SEQUENCE)"** 按钮。
2.  右侧终端会开始接收来自异世界的数据流。
3.  生成完成后：
    *   **复制**: 点击上方 Copy 图标。
    *   **下载**: 点击 Download 图标保存为 `.txt` 文件，作为你的“呈堂证供”。

## 💡 第三章：大份配方 (Recipes)

如何制造顶级的“大份”？这里有几个经过验证的配方：

#### 配方 A：赛博祥子
*   **题材**: Cyberpunk
*   **角色**: 祥子（送外卖的），虎妞（义体医生）
*   **冲突**: 祥子的外骨骼机甲电池被偷了
*   **风格**: 老舍 + 银翼杀手

#### 配方 B：物理学修仙
*   **题材**: Fantasy
*   **角色**: 爱因斯坦（大乘期修士），牛顿（炼丹师）
*   **高潮**: 爱因斯坦祭出“相对论法宝”，扭曲了时空，让牛顿的苹果往天上掉。
*   **风格**: 仙侠网文风

#### 配方 C：发疯文学
*   **题材**: Drama
*   **Tone**: Hysterical (歇斯底里 - 手动输入)
*   **风格样本**: 粘贴一段“发疯文学”段子。
*   **内容**: 随便填，AI 会自动发疯。
